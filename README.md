# n-e.react
```
git clone git@github.com:Memovor/n-e.react.git
```

## 安装依赖

```
yarn install
```

### 项目启动

```
yarn start
```

### 项目打包

```
yarn build
```

### 规范检查

```
yarn lint
```

### mobx 最佳实践

[文档](https://mobx.js.org/react-integration.html)
