## 项目介绍

```
基于React + TypeScript实现类似TodoList的一个工具，状态通过Mobx进行管理
```

### 安装依赖

```
yarn install
```

### 项目启动

```
yarn start
```

### 项目打包

```
yarn build
```

### 规范检查

```
yarn lint
```
